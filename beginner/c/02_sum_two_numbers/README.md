# Challenge 02: Sum Two Numbers

**Difficulty**: Beginner  
**Language**: C  
**Topics**: Functions, Parameters, Return values, Arithmetic

## Description

Write a function that takes two integers and returns their sum.

## Requirements

- Create a function `int sum(int a, int b)` that returns the sum of a and b
- Handle both positive and negative numbers

## Function Signature

```c
int sum(int a, int b);
```

## Examples

```c
sum(5, 3) → 8
sum(-2, 7) → 5
sum(0, 0) → 0
sum(-5, -3) → -8
```

## Skills Practiced

- Function parameters
- Return values
- Integer arithmetic
- Basic math operations

## Testing

Your solution will be tested with various integer combinations including:
- Positive numbers
- Negative numbers
- Zero
- Large numbers
