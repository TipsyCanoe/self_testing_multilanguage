# Challenge 01: Hello World

**Difficulty**: Beginner  
**Language**: C  
**Topics**: Basic I/O, printf

## Description

Write a C program that prints "Hello, World!" to the console.

## Requirements

- Create a function `void print_hello(void)` that prints "Hello, World!" followed by a newline
- The main function should call this function

## Function Signature

```c
void print_hello(void);
```

## Example

```
Output: Hello, World!
```

## Skills Practiced

- Basic C syntax
- Function definitions
- printf usage
- Standard output

## Testing

Run the test suite:
```bash
gcc solution.c -o solution
./solution
```

Expected output should be exactly: `Hello, World!`
