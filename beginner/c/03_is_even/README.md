# Challenge 03: Is Even

**Difficulty**: Beginner  
**Language**: C  
**Topics**: Conditionals, Modulo operator, Boolean logic

## Description

Write a function that determines if a number is even.

## Requirements

- Create a function `int is_even(int n)` that returns 1 (true) if n is even, 0 (false) otherwise
- Use the modulo operator (%)

## Function Signature

```c
int is_even(int n);
```

## Examples

```c
is_even(4) → 1
is_even(7) → 0
is_even(0) → 1
is_even(-2) → 1
is_even(-3) → 0
```

## Skills Practiced

- Conditional logic
- Modulo operator
- Boolean values in C
- Even/odd determination

## Testing

Your solution will be tested with:
- Positive even numbers
- Positive odd numbers
- Zero
- Negative even numbers
- Negative odd numbers
