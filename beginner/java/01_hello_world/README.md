# Challenge 01: Hello World

**Difficulty**: Beginner  
**Language**: Java  
**Topics**: Classes, Methods, Return values

## Description

Write a Java method that returns the string "Hello, World!".

## Requirements

- Create a method `helloWorld()` that returns the string "Hello, World!"
- Method should be static and public
- Return type should be String

## Method Signature

```java
public static String helloWorld()
```
